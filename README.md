# Traffic Light Project
This is a group project aiming to simulate real-life traffic light systems and pedestrian walkways. 
The project uses mbeds, as well as, LEDs and pushbuttons to recreate a 4 way intersection with pedstrian walkways. 
The final code runs for a total of 28 seconds whichis in line with the UK traffic Light time sequence.
## Usage
This code reqires a physical system to work. The code is originally written in Keil studio code to run on an Mbed.
## Limitations
The system currently features code intializing only one pedstrain walkway into the system.
An ideal system would feature 4 walkways
